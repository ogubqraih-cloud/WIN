# 漣 — サイトマップ

参考サイトの構成を踏襲しつつ、22室の隠れ家にふさわしい規模に圧縮した。
URLは英語スラッグに統一（実用的・国際対応）。

## 全ページ一覧

```
/                                トップページ
│
├─ /about/                       漣について（コンセプト・島・建築）
│
├─ /rooms/                       客室一覧
│   ├─ /rooms/sazanami/          漣  -sazanami-（最上級スイート）
│   ├─ /rooms/nagi/              凪  -nagi-（海側スイート）
│   ├─ /rooms/migiwa/            汀  -migiwa-（海側デラックス）
│   └─ /rooms/tana/              棚  -tana-（島側）
│
├─ /dining/                      ダイニング総合
│   ├─ /dining/ushio/            潮  -ushio-（メインダイニング）
│   ├─ /dining/asage/            朝餉の間  -asage-no-ma-（朝食）
│   └─ /dining/nagi-no-ma/       凪の間  -nagi-no-ma-（ラウンジ）
│
├─ /facilities/                  施設（水鏡・潮の湯・文の間・舟屋・手当ての間）
│
├─ /experience/                  過ごし方（5つの体験）
│
├─ /access/                      アクセス
│
├─ /news/                        お知らせ一覧
│   └─ /news/{slug}/             記事詳細
│
├─ /contact/                     予約・お問い合わせ
│
└─ /privacy/                     プライバシーポリシー（簡素）
```

## 参考サイトとの対比

| 参考サイト | 漣 | 判断 |
| --- | --- | --- |
| `/about/` | `/about/` | 採用 |
| `/guestrooms/` | `/rooms/` | URL簡素化 |
| `/guestrooms/details/` | 各室個別ページ | 漣は4タイプを個別URLに |
| `/restaurant/` | `/dining/` | URLとブランドの整合 |
| `/restaurant/breakfast/` 等 | `/dining/asage/` 等 | 命名を漣の和名に |
| `/lounge/` | `/dining/nagi-no-ma/` | ダイニング配下に統合 |
| `/banquet/` | （なし） | 22室の隠れ家には不要 |
| `/facilities/` | `/facilities/` | 採用 |
| `/activities/` | `/experience/` | 名称変更 |
| `/access/` | `/access/` | 採用 |
| `/news/` | `/news/` | 採用 |
| `/contact/`, `/faq/` | `/contact/` に統合 | FAQはコンタクト下部に |
| 各種規約・ポリシー | `/privacy/` のみ | 架空サイトのため簡素化 |

## グローバルナビ（PC）の表示順

```
[ロゴ]   漣について   客室   ダイニング   施設   過ごし方   アクセス   お知らせ   [JP/EN]   [予約 →]
```

- 「予約」ボタンは右端固定、視覚的優先度を最上位に。
- 「お問い合わせ」はナビには出さず、フッターから誘導。
- 「漣について」を最左にして、ブランドの入口として位置づける。

## URLルール

- すべて小文字、スラッシュ終端あり。
- 単語区切りはハイフン（`nagi-no-ma`）。
- 物理ファイル：HTMLは `index.html` をディレクトリに置く形式（クリーンURL）。

```
/rooms/sazanami/index.html  → URL: /rooms/sazanami/
```

ローカル開発で `file://` で見るときは `index.html` 直接アクセスでも動く構造にする（拡張子省略のためサーバ設定に頼らない）。
