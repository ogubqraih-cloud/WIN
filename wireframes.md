# 漣 — ワイヤーフレーム

各ページのセクション構成と意図を、テキストベースで記述。
詳細なコピーは Phase 4 で詰める。ここでは **「何のセクションが・どの順番で・どんな見え方で」** を確定する。

凡例：
- `■` = フルブリード画像／動画
- `□` = テキスト中心ブロック
- `▢` = 画像＋テキストの並列ブロック
- `▤` = カード／グリッド
- `↘` = 次セクションへの誘導（"Scroll" / "詳しく" / 矢印）

## 1. トップページ `/`

最も重要なページ。世界観のすべてを15秒で伝える。

### セクション構成
```
■ 1. ヒーロー（フル画面）
   水面の漣の動画 or 朝凪の島の写真
   中央：「ひと漣の、贅沢。」 + Luxury, in a single ripple.
   下中央：Scroll ↓

□ 2. イントロ（白背景・余白多）
   一行：「水面に、ひと漣の立つ朝がある。」
   段落：3〜4行のブランドコピー
   ↘ 漣について（リンク）

▢ 3. 客室プレビュー
   全画面写真（漣スイートの空間）
   左下：「客室 / Rooms」
   右下：4タイプの名前一覧（漣・凪・汀・棚）
   ↘ 客室一覧へ

▤ 4. ダイニングプレビュー
   3カード（潮・朝餉の間・凪の間）
   PC：横3カラム / SP：縦3スタック
   各カード：写真＋名称＋一行コピー
   ↘ ダイニング詳細へ

▢ 5. 過ごし方プレビュー
   全画面写真（朝凪のボートツアー）
   重ねて：「朝凪の、ひととき。」
   ↘ 過ごし方一覧へ

▤ 6. 施設プレビュー（小）
   5施設を横スクロール or 5カラム小カード
   水鏡・潮の湯・文の間・舟屋・手当ての間
   ↘ 施設詳細へ

■ 7. アクセスティーザー
   フルブリードの島の俯瞰写真
   重ねて：「瀬戸内、凪島へ。」
   ↘ アクセス情報へ

▤ 8. お知らせ（最新3件）
   日付 / カテゴリ / タイトル
   ↘ お知らせ一覧へ

■ 9. クロージング
   月夜の海の写真
   中央：「ひと漣の、贅沢。」
   下：[予約 →] [お問い合わせ →]

[共通フッター]
```

### 演出の重ね方
- ヒーロー：ロード後にロゴと見出しがフェードアップ、画像はゆっくりズーム
- 各セクション：viewport 30%入りでフェードアップ
- イントロから客室への遷移：間に8〜10秒分のスクロール余白を取る

## 3. 客室一覧 `/rooms/`

### セクション構成
```
■ 1. ヒーロー
   「客室 / Rooms」
   背景：客室室内の写真

□ 2. イントロ
   全22室のブランド宣言（4〜5行）

▢ 3. 漣（最上級スイート） — 大きく
   フル幅画像 + 下部にテキスト
   名称・面積・定員・特徴3行
   [詳細を見る →]

▢ 4. 凪 — 中くらい
   左右配置（画像左 / テキスト右）

▢ 5. 汀 — 中くらい
   左右配置（画像右 / テキスト左、互い違い）

▢ 6. 棚 — 中くらい
   左右配置（画像左 / テキスト右）

□ 7. 客室共通の設え
   全室共通のアメニティ・サービス
   テキスト中心、リスト or 短い段落

[予約CTA]
[共通フッター]
```

### 階層表現
- 漣スイートだけ**画像を大きく**（フル幅）して、他は左右配置に。
- 文字サイズも漣スイートだけ大きい見出しを使う。

## ページ間の優先順位（実装順の参照）

1. **共通基盤**（ヘッダー・フッター・タイポ・色）
2. **トップ** — 最重要、世界観の表現
3. **About** — トップに次ぐ世界観の補強
4. **客室一覧 → 客室詳細1ページ**（テンプレート確立）
5. 客室詳細の残り3ページ
6. **ダイニング一覧 → ダイニング詳細1ページ**
7. ダイニング詳細の残り2ページ
8. 施設・過ごし方
9. アクセス
10. お知らせ・コンタクト
11. プライバシー

トップ＋客室1ページ＋ダイニング1ページが完成すれば、世界観の核は伝わる。
